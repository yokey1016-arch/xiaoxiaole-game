# 带薪飞升：下班倒计时

一款手机端竖屏 H5 摸鱼小游戏。玩家在三条办公跑道里左右切换，收集修为和摸鱼值，躲避老板视线，撑满 60 秒即可下班结算。

## 运行方式

用 VS Code Live Server 打开 `index.html` 即可运行。也可以直接在浏览器打开 `index.html` 体验。

## 操作

- 左右滑动：切换工位跑道
- 点击左半屏：向左移动一格
- 点击右半屏：向右移动一格
- 底部按钮：使用“假装认真”
- 调试辅助：`A` / `←` 左移，`D` / `→` 右移，空格释放技能

## 玩法规则

- 初始修为 20，摸鱼值 20，老板警觉 10。
- 接到好道具会提升修为、摸鱼或降低老板警觉。
- 接到坏道具会扣数值或增加老板警觉。
- 15 秒后老板视线开始出现，站在被盯住的跑道会持续增加警觉。
- 最后 15 秒进入下班冲刺，掉落速度和老板压力都会提高。
- 老板警觉达到 100 会立即失败。
- 倒计时归零后，根据最终数值展示不同结局。

## 文件结构

- `index.html`：页面结构
- `style.css`：手机竖屏界面与动效
- `game.js`：游戏状态、掉落、碰撞、技能、结算逻辑
- `README.md`：说明文档
